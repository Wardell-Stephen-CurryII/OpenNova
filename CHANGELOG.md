# Changelog

All notable changes to OpenNova are recorded here.

## [0.4.1] - 2026-07-12

### Changed

- Updated all current user and developer documentation for the Textual-only interactive experience.
- Removed obsolete `opennova tui`, legacy interactive CLI, 17-tool, and copy-overlay documentation.
- Documented in-place message selection, clipboard fallbacks, Windows IME support, and current TUI controls.
- Documented the session picker, complete transcript replay, newest-session continuation, and same-session resume persistence.
- Refreshed command, slash-command, 39-tool, security, extension, SDK, and architecture references.
- Marked development plans 01-13 as archived design records.

### Fixed

- Updated the generated project-guide command example so it no longer recommends the removed `opennova tui` command.

## [0.4.0]

- Added the workbench TUI, stronger Plan/Skills/security foundations, session v2 persistence, SDK events, checkpoints, plugins, automation, diagnostics, and process sandbox support.
